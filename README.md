# wikitextify

Make [Scrum of scrums](https://www.mediawiki.org/wiki/Scrum_of_scrums) meeting
notes beautiful.

## Usage

Copy Scrum of Scrums meeting notes into file named `input.txt`.

```sh
bundle exec ruby bin/wikitextify.rb > output.txt
```
